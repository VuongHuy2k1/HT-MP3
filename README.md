# HT-MP3

This is a graduation thesis project at the University of Transport

## Installation
Use the package manager [NPM](https://www.npmjs.com/) to install.

```bash
cd .\Server\
npm i
cd .\Client\
npm i
```

## Run

Server run at port: 8989

Client run at port: 3000

## Usage

```bash
# Server
cd .\Server\
npm start

# Client
cd .\client\
npm start
```
